# Expense Tracker 💰

A Python command-line application for recording and managing daily expenses.

The application allows users to add expenses, view saved records, and calculate the total amount spent.

## Features

- Add a new expense
- View all recorded expenses
- Calculate total expenses
- Validate expense amount input
- Store data using file handling

## Concepts Used

- Python
- Functions
- Loops
- Conditional Statements
- Lists
- File Handling
- String Manipulation
